# Milovana EOS Editor instructions

## Tips

### Editor

EOS editor can be found at: https://milovana.com/eos/editor/teases

### Copy/Inspect other teases

You can copy other EOS teases by changing their url (change showtease to geteosscript)

For example, this:
https://milovana.com/webteases/showteas ... d4a4294fc7
becomes this:
https://milovana.com/webteases/geteossc ... d4a4294fc7eteosscript

You then can upload the json in the EOS editor (via the backup function).
